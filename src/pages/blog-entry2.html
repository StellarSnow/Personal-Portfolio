Brief Introduction to HTML and CSS

Dane Dwarka published on March 6, 2022

This is a brief Introduction to HyperText Markup Language (HTML) and Cascading Style Sheets (CSS) for beginners.

Every page on the Internet contains HTML. It may or may not contain CSS. You use HTML to create web pages; you use CSS to make them look nice. Web browsers convert HTML and CSS to web pages.

HTML has:

  • Tags
  • Elements
  • Attributes

You combine these to make web pages.

Tags

Tags are placed around words to create sections on a web page. Line 9 from figure 1 has the paragraph tag. It looks like this: “<p>”. Most tags have an opening and a closing. Closing tags contain a forward slash. The closing tag on line 9 is </p>. Some tags do not have a closing. The image tag on line 8 is an example of this.

Elements

Elements consist of an opening tag, the closing tag, and everything in between. Line 4 on figure 1 is an element. It puts the words “My Cool Page” on a tab in the web browser. Elements can span more than one line; the body element in figure 1 runs from lines 6 to 10.

Attributes

Tags contain attributes. Attributes contain further information about tags. For the example, the Image tag (img) contains an attribute called alt. Setting the alt attribute displays a message to the user in case the image does not load.

CSS

CSS applies special effects to elements. It does background colours, borders, font colours, font sizes, line heights, animation, etc. CSS code goes into another document called a style sheet. It looks different from HTML.
