Colour Blocks

The user selects a colour from the drop down box and then clicks on a square. The square gets coloured with the chosen colour. If the user clicks on a square without choosing a colour then the square is coloured with a random RGB colour.

Colour Blocks is written with HTML, CSS, and JavaScript. The HTML and CSS already existed, but I wrote the JavaScript code for this project.

The JavaScript does the following:

  • Populates the "Please Enter a Color" box with 148 different colours
  • Colours a box with the selected colour
  • Ensures a box cannot be coloured if it is already coloured
